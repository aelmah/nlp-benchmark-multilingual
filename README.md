# nlp-benchmark-multilingual
une comparaison de plusieurs techniques NLP à chaque étape du pipeline afin d'identifier les meilleures approches pour un système multilingue :
- Détection de langue : quelle technique identifie le mieux la langue d'un texte ?
- Vectorisation : quelle représentation capture le mieux la sémantique multilingue ?
- Analyse de sentiment : quel modèle prédit le mieux le sentiment sur des textes en 4 langues ?
